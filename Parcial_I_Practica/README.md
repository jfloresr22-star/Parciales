Descripcion:
Este programa fue realizado en C++ y consiste en crear tres nodos usando memoria dinamica con new.
Primero se define la estructura Nodo que contiene un dato entero y un puntero al siguiente nodo.  
Luego en el main se crean tres nodos dinamicamente y el usuario ingresa un valor para cada uno.
Despues los nodos se enlazan manualmente en el mismo orden en que fueron ingresados.  
Se utiliza un puntero auxiliar para recorrer la lista y mostrar los valores almacenados en cada nodo.
Al final del programa se libera la memoria usando delete para evitar fugas de memoria.
Con este ejercicio se demuestra el uso de punteros, memoria dinamica, enlace de nodos y liberacion correcta de memoria.
